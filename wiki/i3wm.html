i3 Tiling Windows Manager

What is the usage of i3wm?

i3wm, or i3 Window Manager, is a tiling window manager for X11-based systems. It is designed to provide a highly efficient and customizable window management experience. Unlike traditional stacking window managers, where windows are typically overlapped, i3wm arranges windows in a non-overlapping manner, known as tiling.

i3wm is going to save your computer from the burden of having to load all those GUI-savvy including applets, desklets, etc. Thus saving more memory and the best guess its memory consumption is around 3% only!It will serve more power to you and you alone.
